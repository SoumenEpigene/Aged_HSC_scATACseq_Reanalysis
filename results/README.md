# Results

This folder contains the major outputs generated during the scATAC-seq reanalysis.

## Files

### DARs_20k_LR.csv
Differentially accessible regions (DARs) identified between aged and young HSCs using logistic regression.

### Aged_open_DAR_genes.csv
Genes associated with regions showing increased accessibility in aged HSCs.

### Aged_closed_DAR_genes.csv
Genes associated with regions showing decreased accessibility in aged HSCs.

## Summary

- Open DARs: 627
- Closed DARs: 499

Open DARs were enriched for developmental and morphogenesis pathways, whereas closed DARs were enriched for lymphocyte differentiation and immune regulatory programs.
